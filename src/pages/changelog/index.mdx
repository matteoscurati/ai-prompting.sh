# Changelog

Tracked in [Keep a Changelog](https://keepachangelog.com/) format and following
[Semantic Versioning](https://semver.org/). Source of truth lives in
[`CHANGELOG.md`](https://github.com/matteoscurati/aiprompting/blob/main/aiprompting/CHANGELOG.md)
in the package repo.

## [0.1.4] — 2026-05-05

Documentation overhaul + npm publish readiness. No functional changes to the CLI, library, or
Skill.

**Added** — Rewritten README with badges, quickstart, FAQ, examples. New `CONTRIBUTING.md`,
`SECURITY.md`, `CODE_OF_CONDUCT.md` (Contributor Covenant 2.1). Matrix CI on Node 18/20/22.
Tag-triggered release workflow with npm provenance. Issue + PR templates. Weekly Dependabot.
`.editorconfig`. `package.json` metadata (homepage, repository, bugs, publishConfig).
`prepublishOnly` gate.

## [0.1.3] — 2026-05-05

**Added** — Slash command `/aiprompting:improve` exposes 4 backend options:
`--language`, `--audience`, `--token-budget`, `--clarify`. Natural-language fallbacks documented
("in italiano", "non chiedere", "minimale"). Flags always win over fallbacks.

## [0.1.2] — 2026-05-05

**Added** — `.claude/commands/improve.md` shipped with the package. Cross-host installer
`scripts/install-command.js` for Claude Code, Codex CLI, Cursor. `npm run install-command` script.

## [0.1.1] — 2026-05-04

**Fixed** — Italian persona padding leaking into `<objective>`. "Aiutami a / Help me" openers now
nominalize correctly.

**Added** — 7 Italian padding patterns (`PADDING_PATTERNS`). `nominalizeImperative()` helper.
4 regression tests.

## [0.1.0] — 2026-05-03

Initial release.

**Added** — `SKILL.md` with the 8-step decision flow. 5 `references/` files. TypeScript core
(Node 18+, no runtime dependencies): `evaluator.ts` (9-category, 100-point rubric),
`prompt-improver.ts`, `doctor.ts`, `cli.ts`, `index.ts`. `bin/aiprompting` shebang shim.
`agents/openai.yaml` for OpenAI-style hosts. 4 example prompts. Full test suite via `node --test`.
